# AI powered search result summarize

This tool is a very basic implementation of the MCP protocol. It contains the following things

## Tools

### web_search 
Performs web search on a given query. This tool implements duckduckgo search

### save_report

Saves the summary of the results returned by the web_search tool.

### get_date

Fetches the date and time using python's datetime module

## LLM

Uses ChatGPT 4.1 to summarize
